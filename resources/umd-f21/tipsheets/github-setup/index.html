Getting started with GitHub

What is GitHub?
GitHub is a provider of Internet hosting for software development and version control using Git.

Links:

GitHub: github.com
Desktop client: desktop.github.com

Getting started

  1. Download and Install Github desktop client at desktop.github.com
  2. Create an account on github.com
  3. Be sure to make your username something professional, ideally a first/last name combo, like kalanigordon. This will be visible in your URL’s for your projects.

Creating a repository

At the top-left, click Create repository


  1. Fill in a repository name, which will be your username, followed by .github.io, ex. kalanigordon0.github.io
  2. Keep the repository public, and do not check any of the boxes below.
  3. Click Create repository

  4. Once completed, you should see the screen similar to below.

Getting started with GitHub Desktop

Open the GitHub desktop client you’ve installed earlier and go to Preferences under the menu.

Complete the following steps:

  1. Account: Sign in to your Github.com account. Once done, go back to Preferences.
  2. Git: Change the name to your username, and change the email to your email from your account. Make sure the branch selected is master.

On your landing page for the desktop client, you should see a repository with the name you just created.

Setting up your local files for your repository

  1. Somewhere on your computer — Documents, Desktop, etc. [I recommend a documents folder] — create a new folder named GitHub. Remember where you saved this folder.

  1. Back in the desktop client, click Clone at the bottom, and change the Local path to the GitHub folder you just created [note: your GitHub folder will be empty, unlike the screenshot below]
  2. Select the GitHub folder. Click Clone.

Now you’re ready to use GitHub.

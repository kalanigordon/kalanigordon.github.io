Setting up Visual Studio Code for J352

Visual Studio Code is an integrated development environment made by Microsoft. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Links:

code.visualstudio.com

Download the Visual Studio Code desktop application

  1. Download VS Code from code.visualstudio.com and open the application.

  2. At the top left, click the bottom icon, Extensions

  3. Search for and Install the extension: Live Server [Ritwick Dey]

  4. Settings: Go to your settings either at the bottom right cog button, or under Preferences in the menu and change the following settings [note: you can search for these keywords using the search bar at the top, seen below]:

Auto Save — change to afterDelay
Word Wrap — change to on
Bracket pair colorization — change to on
Minimap — disable
Workbench: Startup Editor — change to none

Open your repo

  1. Click on the top icon on the left pane, Explorer, and select Open Folder
  2. Find your GitHub folder on your computer, and open the repository you created earlier.

Save your workspace

  1. Under file, Save workspace as...
  2. Navigate to your GitHub folder [not your repository], and create a new folder named vscode
  3. Save your workspace.
